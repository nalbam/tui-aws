# Documentation Sync Rules

- New directory under `internal/` → document purpose in steering or docs
- New AWS API usage → update `.kiro/steering/aws-sdk.md`
- New tab added → update `.kiro/steering/tab-architecture.md` and `docs/architecture.md`
- Architecture decision → create ADR in `docs/decisions/ADR-NNN-title.md`
- Operational procedure → create runbook in `docs/runbooks/`
- ADR numbering: find highest in `docs/decisions/ADR-*.md`, increment by 1
